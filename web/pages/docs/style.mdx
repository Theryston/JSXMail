# Style

Enhance the visual appeal of your email templates in JSX Mail using our extensive selection of style properties. Tailor each element to fit your design needs with these customizable options:

```tsx
  type ElementStyle = {
    backgroundColor?: string; // Set the element's background color
    border?: string; // Define the style for the entire border
    borderBottom?: string; // Customize the bottom border
    borderBottomColor?: string; // Choose the color for the bottom border
    borderBottomStyle?: string; // Specify the style for the bottom border
    borderBottomWidth?: string; // Set the width of the bottom border
    borderColor?: string; // Select a color for all borders
    borderLeft?: string; // Style the left border
    borderLeftColor?: string; // Color for the left border
    borderLeftStyle?: string; // Style type for the left border
    borderLeftWidth?: string; // Width of the left border
    borderRight?: string; // Design the right border
    borderRightColor?: string; // Color for the right border
    borderRightStyle?: string; // Style type for the right border
    borderRightWidth?: string; // Width of the right border
    borderStyle?: string; // General style for all borders
    borderTop?: string; // Style the top border
    borderTopColor?: string; // Color for the top border
    borderWidth?: string; // Set the width for all borders
    margin?: string; // Apply margin around the element
    marginBottom?: string; // Define bottom margin
    marginLeft?: string; // Set left margin
    marginRight?: string; // Apply right margin
    marginTop?: string; // Set top margin
    color?: string; // Choose the text color
    display?: 'block' | 'inline-block' | 'inline'; // Decide the display type
    font?: string; // Use font shorthand property
    fontFamily?: string; // Set the font family
    fontSize?: string; // Define the font size
    fontWeight?: string; // Choose the font weight
    letterSpacing?: string; // Space out characters
    lineHeight?: string; // Set the line height
    listStyleType?: string; // Bullet style for lists
    padding?: string; // Padding inside the element
    paddingBottom?: string; // Bottom padding
    paddingLeft?: string; // Left padding
    paddingRight?: string; // Right padding
    paddingTop?: string; // Top padding
    textAlign?: string; // Align text
    textDecoration?: string; // Add text decoration
    textIndent?: string; // Indent text
    textTransform?: string; // Transform text case
    verticalAlign?: string; // Align vertically
    width?: string; // Set the element's width
    height?: string; // Set the element's height
  }
```

Incorporate these styles directly into your [JSX Mail tag](/docs/tags) by setting the style property with your desired style object.

```tsx
<a style={{ color: 'red' }}>Hello</a>
```

Each property offers you the flexibility to craft the exact look and feel for your email templates, making your emails not just communicative but also visually compelling.
