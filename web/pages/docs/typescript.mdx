# TypeScript

TypeScript offers significant enhancements to your projects, including improved maintainability and error-checking capabilities. Fortunately, integrating TypeScript into your JSX Mail projects is straightforward and efficient.

### Steps to Integrate TypeScript with JSX Mail

<Steps>

### Initial Setup

Ensure TypeScript is installed and configured in your project before initializing JSX Mail with the command `jsxm init`.

### TypeScript Configuration Check

When initializing, JSX Mail's CLI will look for a `tsconfig.json` file in your project. If found, JSX Mail will automatically apply the `tsconfig.json` settings for it works with JSX Mail.

### Automatic JSX Typing

Post-configuration, JSX Mail automatically sets up JSX typing for your TypeScript project, streamlining the development process.

</Steps>

For further details and guidance on getting started, refer to the [JSX Mail Quickstart Guide](/docs). This guide provides comprehensive information on the basics and advanced usage of JSX Mail, ensuring a smooth integration with TypeScript in your projects.
