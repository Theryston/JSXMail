The prepare command is used to prepare all your project to render your email templates.

This command is very important because it will check if everything is set up correctly and if there are any errors in your project. and also upload your files to the JSX Mail Cloud.

To run the prepare command, you can use the following command:

```sh
yarn jsxm prepare
```

If you want to know more about the prepare command, you can check the [prepare page](/framework/learning/rendering#preparing)
